Radio Frequencies

I'm not too sure how good these are at helping you spot trains, as I've yet to actually use them, but I thought I might add it. You should be able to use just any generic UHF radio to listen to the frequencies in the 400mhz range, but if you don't have one I would personally recommend the Baofeng UV5R. While it is not the best, it is certaily cheap, weighing in at around 38 AUD.

Here's some links:

  • https://www.aliexpress.com/w/wholesale-baofeng-uv5r-australia.html
